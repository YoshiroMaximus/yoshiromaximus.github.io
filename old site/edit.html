✏️ Editing Your Portfolio

Welcome to the editing guide! Follow the steps below to customize your portfolio and keep it up to date with your latest work. 🎨

🔧 Steps to Edit

  • Step 1: Open the HTML files 📝

    Open the HTML files of your portfolio in a text editor. We recommend using Visual Studio Code or Sublime Text for a better editing experience.

  • Step 2: Update content 📄

    Update the content within the <body> tags. For example, you can modify the text in the <h1> and <p> tags to change the headings and paragraphs.

  • Step 3: Save changes 💾

    After making your changes, save the HTML files. Ensure you save them with the correct file extensions, such as .html.

  • Step 4: View your changes 👀

    Open your browser and navigate to the HTML file to see your changes in action. If you're using a local development server, refresh the page to view updates.

💡 Tips and Tricks

  • Use meaningful file names 📁: Name your files clearly to easily identify their content and purpose.
  • Keep it simple 🧹: Maintain clean and well-organized code for easy future updates and maintenance.
  • Test regularly 🔄: Frequently test your website in different browsers to ensure compatibility and functionality.
